[[payara-maven-plugins]]
= Payara Maven Plugins

The Payara Maven Plugins provide seamless integration with Maven, enabling developers to start, stop, and deploy Jakarta EE applications on Payara Server and Payara Micro. These plugins streamline development, support automated deployment, and enhance workflow efficiency.

[[plugins]]
== Plugins
* xref:Technical Documentation/Ecosystem/Project Management Tools/Maven Plugin/Payara Server Maven Plugin.adoc[Payara Server Maven Plugin] - The Payara Server Maven Plugin allows developers to start, stop, and deploy applications to Payara Server directly from Maven, streamlining the development and deployment process.

* xref:Technical Documentation/Ecosystem/Project Management Tools/Maven Plugin/Payara Micro Maven Plugin.adoc[Payara Micro Maven Plugin] - The Payara Micro Maven Plugin enables seamless integration with Payara Micro, providing goals to start, stop, and package lightweight, microservice-based Jakarta EE applications.
