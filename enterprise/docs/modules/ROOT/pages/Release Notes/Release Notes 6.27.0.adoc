= Release notes - Payara Platform Enterprise 6.27.0

== Supported APIs and Applications

* Jakarta EE 10
* Jakarta EE 10 Applications
* MicroProfile 6.1


== Improvements

* [FISH-8455] Add support for collecting logs, heap dumps, thread dumps, and jvm reports for a remote SSH instance from the DAS

* [FISH-8498] Add support for parsing local instance port from domain.xml

* [FISH-9754] Diagnostic tool - IP address in the server.log

* [FISH-11054] Add support for MicroProfile OpenAPI 3.1 Bean Validation

* [FISH-11197] [Community Contribution - https://github.com/pzygielo[pzygielo]] Align Cluster Deprecation Message With Product Versions

== Bug Fixes

* [FISH-8795] Fix `java.lang.NullPointerException` caused by redeployment of an application that modifies the OpenAPI document

* [FISH-10996] Fix Javadoc Generation

* [FISH-11029] Fix IMQ and IMQJMX Versions

* [FISH-11223] [Community Contribution - https://github.com/pzygielo[pzygielo]] Fix Cannot find the resource bundle for the name `com.sun.logging.enterprise.system.container.ejb` for class `org.glassfish.ejb.persistent.timer.TimerState` Error
